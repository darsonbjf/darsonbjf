## 📌 Projetos em destaque (públicos)
- **observability-stack** — Compose com OTEL Collector → Prometheus, Loki, Tempo e Grafana (dashboards prontos).
- **gpo-scripts** — PowerShell para impressão P&B, certificados, Firefox/Edge/Chrome (com logs e rollback).
- **n8n-flows** — Fluxos WhatsApp + Redis (JSONs mascarados + README de arquitetura).
- **otel-node-template** — Boilerplate Node.js/TS com OTEL gRPC, /metrics (prom-client) e exemplos de traces.
- **docker-swarm-secrets-template** — Padrão de deploy com secrets/healthchecks e entrypoint.
